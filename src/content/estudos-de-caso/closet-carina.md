### 1. O Desafio do Espaço e a Engenharia Anti-Erro
O projeto exigia a criação de dois closets em um espaço extremamente compacto (3,5 x 1,6 metros). Um dos lados do closet exigia uma porta de correr majestosa, indo do piso ao teto, cobrindo um vão de 2,4 metros de altura.

**A Solução Técnica:** Portas de MDF com essa altura possuem alto risco de encurvamento. Para garantir a estabilidade a longo prazo, o projeto executivo previu a usinagem e instalação de **desempenos metálicos** embutidos nas portas. Sem esse detalhe de engenharia, as portas empenariam em poucos meses.

### 2. O Segredo da Instalação Perfeita (Prevenção de Desnível)
Para absorver os desníveis comuns de piso, parede e teto das construções civis, o projeto foi estruturado com:
*   **Rodapé Separado:** Uma base de nivelamento independente que permite ao montador ajustar a estrutura ao prumo do piso antes de apoiar os módulos pesados.
*   **Fechamentos Estratégicos:** Áreas de escape projetadas nas laterais e no teto.

### 3. Especificações Técnicas e Ferragens
Para o aproveitamento volumétrico, desenhamos gavetões com frentes largas, chegando a 715mm. 
**A Escolha da Ferragem:** Nestas proporções, o uso de corrediças telescópicas comuns é um erro fatal. Para resolver isso, o projeto especificou o uso de **corrediças ocultas com sistema de sincronização**, garantindo que a gaveta abra de forma perfeitamente paralela.

### 4. Eficiência de Chapa e Dados de Produção (Cortecloud)
O projeto foi integralmente parametrizado no software SketchUp (plugin Hellomob) e enviado para usinagem automatizada.
*   **Índice de Eficiência de Chapa (IEC):** 89% de aproveitamento.
*   **Materiais Utilizados:** 4 chapas de MDF 6mm Branco (Sudati) e 9 chapas de MDF 18mm Branco (Berneck).
*   **Acabamento:** Fita de borda Rehau 0.4 x 22mm.
*   **Investimento na Central de Serviços:** R$ 5.616,52 (Material + Corte e Furação CNC).
